# موقع شركة حميد للأدوية

هذا المشروع عبارة عن موقع ويب ثابت يحتوي على صفحات:
- `index.html` (الرئيسية)
- `about.html` (من نحن)
- `contact.html` (تواصل معنا)
- `delivery.html` (التوصيل)
- `register.html` و `signup.html` (المصادقة)

## بنية المجلدات
```
/HAMID
├── css/
│   └── style.css
├── js/
│   └── script.js
├── index.html
├── about.html
├── contact.html
├── delivery.html
├── register.html
├── signup.html
├── README.md
```

### ملاحظات نشر
- للتشغيل محليًا: يمكن استخدام خادم بسيط مثل `python -m http.server` من جذر المشروع.
- ارفع محتويات المجلد إلى أي استضافة ويب ثابتة (GitHub Pages, Netlify, Vercel, الخ).
- تأكد أن `index.html` في الجذر لأن بعض الخوادم تعتمدها كصفحة افتراضية.

## تطوير
- أساليب CSS داخل `css/style.css`، وملف JavaScript متواجد في `js/script.js`.
- لا تعتمد على إطار عمل؛ تصميم بسيط بالعربية يدعم اتجاه الكتابة من اليمين لليسار.
